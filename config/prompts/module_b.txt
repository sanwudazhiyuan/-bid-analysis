## 提取要求
1. 识别投标人必须满足的资格条件（营业执照、注册资本、经营范围等）
2. 提取禁止情形（如被列入失信名单、处于被责令停业状态等）

4. 提取联合体投标要求（是否允许、条件等）
5. 提取需要提供的资格证明材料清单

## 注意事项
- 如果某些信息在文档中找不到，省略该 section
- 资格条件可能散落在采购公告、投标人须知、资格条件专章等多处，需全部汇总
- 注意区分"必须满足"的硬性条件和"优先考虑"的加分项
- 保持 JSON 格式严格，不要在 JSON 外添加任何文字说明
- id 必须以 "B" 开头，如 B1, B2, B3...
